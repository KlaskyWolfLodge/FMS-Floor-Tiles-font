FONTLOG for the FMS Floor Tiles font

This file provides detailed information on the FMS Floor Tiles
font Software.

This information should be distributed along with the FMS Floor
Tiles font and any derivative works.

Basic Font Information

FMS Floor Tiles is inspired by every floor tile at Frelinghuysen
Middle School.

Designed by Klasky Wolf Lodge.

FMS Floor Tiles is a Unicode typeface family that supports languages
that use the Latin script and its variants, and could be expanded to
support other scripts.

More specifically, this release supports the following
Unicode ranges: Latin-1 and Latin-2

To contribute to the project contact Klasky Wolf Lodge
at klaskywolflodge@gmail.com
